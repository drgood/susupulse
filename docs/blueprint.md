# **App Name**: SusuPulse

## Core Features:

- Susu Dashboard & Navigation: A dynamic dashboard displaying key metrics like total members, total collected funds, admin profit, and defaulter count. Features horizontally scrollable 'pill' tabs for easy switching between different active Susu groups, with an active group visually highlighted.
- Susu Group Management: Functionality to create new Susu groups by inputting parameters such as name, daily contribution, admin fee, maximum members, total Susu duration, flexible payment options (e.g., daily, weekly with deadlines), a fixed cash-out amount, and MoMo details. Allows editing existing Susu group details and provides a confirmation flow for deleting a Susu group.
- Member & Payment Tracking: Displays a comprehensive list of members for each Susu group, showing name, position, payment progress (days paid), and a visual grid to mark payments. The system accommodates flexible payment frequencies (e.g., marking 7 days as paid for a weekly payment). Enables recording payments (including multi-day selections) and marking members as 'Cashed Out'.
- Susu Financial Summary: Provides a detailed financial breakdown for each Susu, including calculated total pool, expected member payouts (based on the defined cash-out amount), and total admin earnings. All calculations for 'netDaily', 'totalCollected', 'totalProfit', and 'defaulters' are computed dynamically, adapting to the flexible payment and cash-out structures.
- AI Admin Insight Tool: Leverages generative AI to provide proactive insights for the Susu administrator, such as identifying potential defaulters, suggesting optimized communication strategies for group members, or proposing ideal payout cycles based on historical payment data, flexible payment options, and group parameters.
- Shareable Payment Status Tool: Generates a pre-formatted message containing group information, flexible payment instructions (daily/weekly amounts, deadlines), and individual member payment progress, optimized for sharing via platforms like WhatsApp. Includes a 'Copy to Clipboard' button for convenience.

## Style Guidelines:

- Light theme for clarity and readability. Primary color: a vibrant, fresh green (#4FD87D) symbolizing growth and vitality. Background color: a very light, almost-white green (#ECF6EF) providing a clean canvas with a subtle thematic hint. Accent color: an energetic, darker yellowish-green (#68E60A) for highlighting key actions and information.
- 'Inter' (sans-serif) for both headlines and body text. This choice promotes a modern, clean, and highly readable interface suitable for a functional management application.
- Utilize a consistent set of clean, modern SVG icons that align with financial management, communication, and administrative tasks, ensuring clarity and intuitiveness. Icons for member status (Active, Behind, Cashed Out) should visually map to green, red, and gold, respectively.
- Adopt a mobile-first, responsive design approach featuring UI elements such as rounded cards (border-radius: 10-14px) with subtle shadows for depth. Maintain compact spacing between elements to optimize content display without visual clutter.
- Implement subtle, smooth UI transitions for state changes, navigation between Susu groups, and marking payments. Integrate micro-interactions and visual feedback animations for user actions, enhancing the overall user experience.
